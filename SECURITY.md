# Security Policy

If you discover a security issue related to this repository (e.g., a flaw in a detection rule that could be bypassed, or a credential/secret accidentally committed):

- **Do not open a public issue.**
- Contact: rezwanbari@gmail.com, or reach out via [LinkedIn](https://www.linkedin.com/in/rezwanulbari/).

This repository contains reference architecture, detection logic, and documentation. No production credentials, customer data, or proprietary client information should ever be present here — if you find any, please report it immediately using the contact above so it can be removed.
